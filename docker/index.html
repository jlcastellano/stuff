Manual Completo de Docker

Docker es una plataforma de virtualización a nivel de sistema operativo que permite crear, distribuir y ejecutar aplicaciones en contenedores (una unidad estándar de software que empaqueta código y todas sus dependencias para que la aplicación se ejecute de manera rápida y confiable en diferentes entornos).
