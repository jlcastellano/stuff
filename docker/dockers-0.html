0. Instalación de Docker

0.1 Instalación en Ubuntu

Requisitos Previos

  • Ubuntu 20.04 LTS o superior
  • Usuario con privilegios sudo
  • Terminal abierta
  • Conexión a Internet

Pasos de Instalación

# 1. Actualizar el sistema
sudo apt update
sudo apt upgrade -y

# 2. Instalar paquetes previos
sudo apt install -y apt-transport-https ca-certificates curl gnupg lsb-release

# 3. Añadir clave GPG oficial de Docker
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg

# 4. Configurar repositorio estable
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

# 5. Instalar Docker Engine
sudo apt update
sudo apt install -y docker-ce docker-ce-cli containerd.io

# 6. Añadir usuario al grupo docker (opcional, para ejecutar sin sudo)
sudo usermod -aG docker $USER

Verificación de la Instalación

# Verificar versión de Docker
docker --version

# Verificar instalación con contenedor de prueba
docker run hello-world

Instalación de Docker Compose

# 1. Descargar Docker Compose
sudo curl -L "https://github.com/docker/compose/releases/latest/download/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose

# 2. Aplicar permisos de ejecución
sudo chmod +x /usr/local/bin/docker-compose

# 3. Verificar instalación
docker-compose --version

0.2 Instalación en Windows

Requisitos del Sistema

  • Windows 10/11 Pro, Enterprise o Education
  • Virtualización habilitada en BIOS
  • WSL 2 instalado
  • Mínimo 4GB RAM

Instalación de WSL 2 (Prerequisito)

# 1. Abrir PowerShell como administrador y ejecutar:
wsl --install

# 2. Habilitar el subsistema de Windows para Linux
dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

# 3. Habilitar características de máquina virtual
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

# 4. Reiniciar el sistema

# 5. Establecer WSL 2 como versión predeterminada
wsl --set-default-version 2

Verificación en Windows

# Abrir PowerShell y verificar versión
docker --version

# Probar instalación
docker run hello-world
