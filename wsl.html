Pasos para instalar WSL2 en Windows 10/11

  1. Abre PowerShell como administrador y ejecuta el siguiente comando para habilitar la característica de Subsistema de Windows para Linux:
    dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
  2. Habilita la característica de Máquina Virtual ejecutando el siguiente comando:
    dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
  3. Reinicia tu computadora para completar la instalación de WSL y la característica de Máquina Virtual.
  4. Descarga el paquete de actualización del kernel de Linux de WSL2 para x64 desde el siguiente enlace:
    Descargar paquete de actualización del kernel de Linux
  5. Ejecuta el paquete de actualización descargado (wsl_update_x64.msi) y sigue las instrucciones del instalador.
  6. Abre PowerShell y ejecuta el siguiente comando para establecer WSL2 como la versión predeterminada:
    wsl --set-default-version 2
  7. Instala tu distribución de Linux preferida desde la Microsoft Store, como Ubuntu, Debian, o Kali Linux.
  8. O desde el terminal usando:
    wsl --list --online
  9. Y luego:
    wsl --install -d NombreDistribucion
  10. Para ver qué distribuciones tienes instalaladas:
    wsl --list --verbose
  11. Abre la distribución de Linux instalada y completa la configuración inicial, creando un nombre de usuario y contraseña.

¡Listo! Ahora tienes WSL2 instalado en tu sistema Windows 10 y puedes disfrutar de un entorno Linux completamente integrado.
