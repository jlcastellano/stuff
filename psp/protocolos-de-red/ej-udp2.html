Ejercicios de Protocolos de Red

UDP

2. Sistema de Monitoreo de Sensores

Descripción:

Implementar un sistema distribuido que simule el monitoreo de sensores de temperatura usando UDP.

Requisitos:

  • Crear múltiples instancias de sensores (clientes) que:
    • Generen lecturas aleatorias de temperatura entre -10°C y 40°C
    • Envíen datos cada 5 segundos al servidor
    • Incluyan un identificador único en cada mensaje
  • El servidor debe:
    • Mantener un registro de los últimos 10 valores por cada sensor
    • Calcular y mostrar la temperatura promedio por sensor
    • Detectar y notificar cuando un sensor deja de enviar datos
    • Generar alertas si la temperatura sale de un rango establecido
