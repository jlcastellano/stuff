Ejercicios de Protocolos de Red

TCP

3. Servidor de Archivos

Descripción:

Implementar un servidor de archivos con funcionalidades básicas de almacenamiento y recuperación usando TCP.

Requisitos:

  • El servidor debe:
    • Manejar múltiples conexiones concurrentes usando hilos
    • Mantener una estructura de directorios organizada
    • Implementar un sistema de logs para todas las operaciones
    • Verificar la integridad de los archivos transferidos
  • Los clientes deben poder:
    • Listar archivos disponibles en el servidor
    • Subir archivos de cualquier tipo
    • Descargar archivos existentes
    • Recibir información sobre el progreso de las transferencias
  • Implementar un protocolo de comunicación que incluya:
    • Comandos para cada operación (LIST, UPLOAD, DOWNLOAD)
    • Respuestas con códigos de estado
    • Manejo de errores y excepciones
