Manual de Uso de Markdown en Jupyter

Introducción

Markdown es un lenguaje de marcado ligero que se utiliza para formatear texto. En Jupyter, puedes usarlo para crear celdas de texto con formato.

Encabezados

# Encabezado 1
## Encabezado 2
### Encabezado 3

Estilos de Texto

  • Negrita: Usa **texto** o __texto__.
  • Cursiva: Usa *texto* o _texto_.
  • Tachado: Usa ~~texto~~.

Listas

Listas No Ordenadas

- Elemento 1
- Elemento 2
  - Sub-elemento

Listas Ordenadas

1. Primer elemento
2. Segundo elemento
   1. Sub-elemento
