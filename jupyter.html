Pasos para instalar JupyterLab en WSL y acceder desde Windows

  1. Abre la terminal de WSL (por ejemplo, Ubuntu).
  2. Actualiza la lista de paquetes ejecutando el siguiente comando:
    sudo apt update
  3. Instala Python y pip si aún no los tienes instalados:
    sudo apt install python3 python3-pip
  4. Instala JupyterLab utilizando pip:
    pip3 install jupyterlab
  5. Configura JupyterLab para permitir conexiones desde cualquier dirección IP. Crea un archivo de configuración ejecutando:
    jupyter lab --generate-config
  6. Abre el archivo de configuración con un editor de texto:
    nano ~/.jupyter/jupyter_lab_config.py
  7. Descomenta y modifica las siguientes líneas para permitir conexiones desde cualquier dirección IP: 
    c.ServerApp.ip = '0.0.0.0'
c.ServerApp.port = 8888
c.ServerApp.open_browser = False
c.LabApp.expose_app_in_browser = True
  8. Guarda los cambios y cierra el editor (Ctrl+X, luego Y y Enter).
  9. Inicia JupyterLab ejecutando:
    jupyter lab --no-browser
  10. Copia la URL que aparece en la terminal, que debería ser similar a:
    http://localhost:8888/?token=...
  11. En Windows, abre un navegador web y pega la URL copiada.
  12. Para acceder a los archivos ubicados en Windows desde JupyterLab, navega hasta el directorio donde se encuentran montadas las unidades de Windows en WSL. Por ejemplo:
    /mnt/c/Users/<username>/Documents

¡Listo! Ahora puedes utilizar JupyterLab en WSL y acceder a él desde un navegador web en Windows. Además, puedes trabajar con archivos ubicados en el sistema de archivos de Windows desde JupyterLab en WSL.
