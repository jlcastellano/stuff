Manual Detallado de Comandos WSL

1. GESTIÓN BÁSICA DE WSL

Instalación y configuración:

wsl --install # Instala WSL con Ubuntu por defecto
wsl --install -d <Distribución> # Instala una distribución específica
wsl --list --online # Muestra distribuciones disponibles
wsl --list --verbose # Muestra distribuciones instaladas
wsl --set-default <Distribución> # Establece la distribución predeterminada

2. CONTROL DE DISTRIBUCIONES

Operaciones básicas:

wsl -d <Distribución> # Inicia una distribución específica
wsl --terminate <Distribución> # Detiene una distribución
wsl --unregister <Distribución> # Desinstala una distribución
wsl --shutdown # Detiene todas las distribuciones

3. GESTIÓN DE VERSIONES

Comandos de versión:

wsl --set-version <Distribución> 2 # Convierte a WSL 2
wsl --set-default-version 2 # Establece WSL 2 como versión predeterminada
wsl --status # Muestra el estado de WSL

4. INTEROPERABILIDAD WINDOWS-WSL

Ejecución de comandos:

wsl <comando> # Ejecuta un comando Linux desde Windows
wsl.exe --exec <comando> # Ejecuta un comando sin iniciar shell
wsl --user <usuario> <comando> # Ejecuta como usuario específico

5. GESTIÓN DE SISTEMA DE ARCHIVOS

Acceso a archivos:

cd /mnt/c # Accede al disco C: de Windows
cd ~ # Accede al home directory de Linux
\\wsl$ # Accede al sistema de archivos WSL desde Explorer

6. REDES Y PUERTOS

Gestión de red:

wsl --networkingMode default # Configura modo de red predeterminado
wsl --networkingMode bridged # Configura modo de red puente
ip addr show # Muestra direcciones IP en WSL

7. GESTIÓN DE MEMORIA Y RECURSOS

Configuración de recursos:

# Crear/editar archivo .wslconfig en %UserProfile%
[wsl2]
memory=4GB
processors=2

8. IMPORTACIÓN/EXPORTACIÓN

Respaldo y migración:

wsl --export <Distribución> <archivo.tar> # Exporta una distribución
wsl --import <Distribución> <InstallLocation> <archivo.tar> # Importa una distribución

9. SOLUCIÓN DE PROBLEMAS

Comandos de diagnóstico:

wsl --status # Muestra estado general
wsl --version # Muestra versión de WSL
wsl --update # Actualiza el kernel de WSL

10. CONSEJOS ADICIONALES

GUI en WSL:
  • WSL 2 soporta aplicaciones GUI directamente
  • Instalar un servidor X si usas WSL 1
VS Code Integration:
  • Instalar extensión "Remote - WSL"
  • Usar comando "code ." dentro de WSL
Bases de Datos:
  • Los servicios como MySQL, PostgreSQL funcionan normalmente
  • Accesibles desde Windows usando localhost
Docker:
  • Docker Desktop se integra con WSL 2
  • Permite ejecutar contenedores nativamente
