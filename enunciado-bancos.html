Ejercicio: Simulador Básico de Transacciones Bancarias Concurrentes

Objetivo

Desarrollar un programa en Java que simule operaciones bancarias básicas realizadas por múltiples clientes de forma simultánea. El programa debe demostrar el uso de hilos.

Requisitos

1. Clase CuentaBancaria

  • Atributos:
    • Saldo (double)
    • Nombre del banco (String)
  • Constructor que inicialice el saldo inicial y nombre del banco
  • Métodos sincronizados:
    • depositar(int idCliente, double cantidad)
    • retirar(int idCliente, double cantidad)
    • getSaldo()

2. Clase Cliente (extends Thread)

  • Atributos:
    • Referencia a CuentaBancaria
    • ID del cliente (int)
  • Funcionalidad en run():
    • Realizar 3 operaciones aleatorias
    • Cada operación: depósito o retiro de 100 €
    • Espera aleatoria entre operaciones (máx. 1 segundo)

3. Clase SimuladorBanco

  • Crear cuenta con saldo inicial de 500 €
  • Crear 5 clientes
  • Iniciar hilos de clientes
  • Esperar finalización de todos los hilos
  • Mostrar saldo final

Ejemplo de Salida Esperada
